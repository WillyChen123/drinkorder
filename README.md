# Drink Order

A Simple Website on Google App Engine

##Introduction
每個帳號可以透過登入網站及輸入欲訂購之飲料  
協助訂購人統計飲料數量及金額

##Usage

####LocalHost

```
dev_appserver.py drinkorder
```

####Online

```
appcfg.py update drinkorder
```
